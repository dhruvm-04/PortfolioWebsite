# Portfolio

A single-page portfolio website designed as an immersive, interactive profile of my work. The site blends atmospheric visuals, motion-driven UI, and structured storytelling to present experience, projects, skills, and certifications in a modern format.

## Website Concept

The portfolio is built around a split-layout experience:
- A fixed left panel for identity, navigation, and social links
- A dynamic right panel where each section transitions in with smooth motion

The visual language emphasizes depth and ambiance through animated grain, a constellation-style node network, and subtle floating micro-interactions.

## Key Sections

## Home
Introduces my focus areas: practical AI systems, multilingual speech pipelines, and production-oriented full-stack development.

## About
Highlights academic background, technical skills, leadership roles, and a custom animated skills graph with structured parent-child nodes.

## Experience
Presents internship outcomes and an animated certification timeline that includes issue dates, credential issuers, and link buttons for each certification.

## Projects
Showcases major projects as a layered, card-stack interface with animated transitions and navigable progression.

## Design and Interaction Highlights

- Theme-aware experience (light/dark) with consistent icon recoloring
- Animated splash introduction and directional tab transitions
- Smooth right-pane scrolling and section-specific interaction models
- Canvas-driven background network with ambient motion
- Responsive behavior tuned for desktop and mobile layouts

## Tech Stack

- HTML5
- CSS3 (custom properties, layered gradients, keyframe animation)
- Vanilla JavaScript (rendered tab content, interaction logic, animation control)

## Purpose

This website is intended to serve as both a professional portfolio and a design-forward demonstration of frontend interaction quality, presenting technical achievements in a format that is clean, memorable, and engaging.
